# Aplicación de Transporte - Frontend

Este es el frontend de la Aplicación de Transporte, desarrollado en React.

## Requisitos

- Node.js >= 16
- npm o yarn

## Instalación

```bash
cd Frontend/frontend
npm install
```

## Ejecución en desarrollo

```bash
npm run dev
```

La aplicación estará disponible en [http://localhost:5173](http://localhost:5173).


## Estructura

- `src/pages`: Páginas principales de la app.
- `src/services`: Lógica para consumir la API del backend.

## Notas

- No subas archivos `.env` al repositorio.
